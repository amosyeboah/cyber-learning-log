# cyber-learning-log
🚀 Cyber Learning Log

A structured, day-by-day documentation of my cybersecurity journey — from fundamentals to hands-on labs, tools, projects, and certifications.
This repository serves as my learning journal, evidence of skills, and portfolio of practical work as I progress through the Google Cybersecurity Professional Certificate and other security training.

📁 Repository Structure

cyber-learning-log/

│── notes/          # Daily learning notes & summaries

│── labs/           # Hands-on lab exercises & write-ups

│── tools/          # Tool usage guides (Nmap, Wireshark, Burp Suite, etc.)

│── cheatsheets/    # Quick reference sheets (Linux, networking, commands)

│── screenshots/    # Lab evidence & walkthrough screenshots

│── projects/       # Security projects & capstones

│── configs/        # Scripts, configs & automation

│── certificates/   # Earned certificates
