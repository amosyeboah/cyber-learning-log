# Windows Security Tools
The Windows operating system includes, and allows for the use of, powerful tools for maintaining system security, monitoring performance, and diagnosing issues.

Windows Defender (Microsoft Defender Antivirus)
Windows Defender (now primarily known as Microsoft Defender Antivirus) is the built-in, comprehensive security solution that comes with Windows.

Antivirus and Anti-Malware: It provides real-time protection against viruses, malware, spyware, and other threats. It constantly monitors your system and files for malicious activity.


Integrated Security: It's part of the broader Windows Security application, which centralizes access to other critical security features, including:

Firewall & network protection: Manages the Windows Firewall.

Account protection: Features like Windows Hello and dynamic lock.

App & browser control: Uses Microsoft Defender SmartScreen.

Device performance & health: Reports on updates, drivers, and battery life.
